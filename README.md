# 🧱Tetris

Tetris game built with Raylib based on the Tetris on Raylib repo(https://github.com/raysan5/raylib-games/blob/master/classics/src/tetris.c).



## 🏃🏼‍♂️How to run
Clone the repo ->
cd Tetris
-> go run main.go
